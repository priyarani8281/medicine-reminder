## ğŸ“– Usage Guide

### Adding Medicines

1. Navigate to **Medicines** tab
2. Fill in the form:
   - Medicine name (e.g., "Aspirin")
   - Dosage (e.g., "500mg")
   - Type (Tablet, Capsule, Syrup, etc.)
   - Frequency (Once/Twice/Thrice daily)
   - Times (specific hours)
   - Start date & duration
   - Special instructions
3. Click **Add Medicine**

### Viewing Analytics

1. Go to **Analytics** tab
2. View charts:
   - 30-day adherence trend
   - Weekly overview
   - Medicine-wise performance
3. Generate PDF reports

### Managing Reminders

- **Enable notifications** in Settings
- **Adjust reminder advance time** (5-60 minutes)
- **Snooze** reminders when needed
- **Mark as taken** directly from notification
